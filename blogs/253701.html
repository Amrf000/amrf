ubuntu 18下安装memcache

apt-get install memcached libmemcached-tools -y
vi /etc/memcached.conf
如果希望局域网内可以使用需要修改监听ip127.0.0.1 => 0.0.0.0
保存后重启memcache服务
systemctl restart memcached
