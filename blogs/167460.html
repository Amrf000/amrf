修改conf/nginx.conf端口

运行nginx.exe

修改nginx.conf 增加

        location /lua {
			default_type 'text/plain';
			content_by_lua_block {
				ngx.say("Hello,world!")
			}
        }

        location /luadir {
			default_type 'text/plain';
			content_by_lua_file 'lua/hello.lua';
        }

hello.lua 中也只放一个lua打印

nginx.exe -s reload后，就可以看到

然后我们就可以在nginx上愉快的使用lua脚本了.
