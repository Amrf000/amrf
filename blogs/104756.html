项目地址:

https://github.com/ghiscoding/angular-markdown-editor

测试步骤:

  1. 下载工程项目

  2. npm install 安装相关依赖，bootstrap-markdown无法获取先打开package.json注释掉再次运行npm install

  3. https://github.com/toopay/bootstrap-markdown/tags下载release包，并解压到node_modules目录

  4. 使用vscode打开项目目录

  5. 在集成终端中输入ng serve启动项目，可以看到

  6. 不是很喜欢上下结构，改一改

    打开template.componet.html

    修改为
